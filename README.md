# Простая змейка на WPF C# 
___
### Данная программа эмулирует оригинальную змейку.
#### Возможности
* **Поедание кукурузы**
* **Увелечение хвоста**
* **Сталкивание с хвостом и со стенками**
* **Увелечение скорости игры**
* **Перезапуск игры**
* ~~**Просто лютая змея**~~
